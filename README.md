# 0IBSIP
Oasis Infobyte Python Internship Tasks
my collection of Python GUI applications developed as part of the Oasis Infobyte Virtual Internship Program, built using Tkinter.

📁 Projects Overview
1.BMI Calculator:
A sleek Body Mass Index calculator with a modern Deep Space Indigo themed GUI.

Main Features:
1.Input weight in kg or lbs
2.Input height in meters or feet
3.Displays BMI value, category, suggested weight range and height range
4.Smooth color flash animations on result labels
5.Clean dark-themed UI with indigo/violet palette

2.Chat Application:
A real-time chat application built with Python's socket and Tkinter libraries.

Main Features:
1.Multi-client support via socket programming
2.Simple and intuitive GUI chat window
3.Real-time message broadcasting between users
4.Username-based identification
5.Scrollable message history

3.Password Generator(Random Type):
A secure and customizable password generator with GUI.

Main Features:
1.Choose password length
2.Toggle inclusion of uppercase, lowercase, digits and special characters
3.One-click copy to clipboard
4.Instant strength indicator
5.Clean minimal interface
